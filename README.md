# Keypress Package

A nodejs keypress function

## Usage

```
const keypress = require('@mattamorphic/keypress').keypress;

// or
// import keypress from '@mattamorphic/keypress';

(async () => {
  await keypress('Press any key to continue.');
  console.log('Good job!');
  process.exit(0);
})()

```
